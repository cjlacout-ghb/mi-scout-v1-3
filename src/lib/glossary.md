# Glossary of Terms

- **bateador en turno**: El bateador actual al que se le está trackeando la información durante el juego.
- **al bate**: Variable que indica el bateador al que se le está trackeando la información; se puede usar como sinónimo de "bateador en turno".
- **jugador abridor**: aquel que inicia un partido.
- **jugador sustituto**: aquel que entra al juego en reemplazo de un jugador abridor.

These terms are used throughout the application for user guidance and documentation.
